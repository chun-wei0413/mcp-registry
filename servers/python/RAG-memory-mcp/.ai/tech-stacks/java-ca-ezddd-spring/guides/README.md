# 開發指南 (Development Guides)

本目錄包含開發指南，幫助團隊在日常開發中快速查找所需資訊。

## 📁 現有指南

### DATABASE-MIGRATION-GUIDE.md - 資料庫遷移指南
- Flyway 配置與使用
- 資料庫版本控制最佳實踐
- 遷移腳本撰寫規範
- 開發、測試、生產環境配置

### DEVELOPMENT-TOOLS-GUIDE.md - 開發工具與命令指南
- Maven 常用命令參考
- Git 操作指令
- IntelliJ IDEA 設定建議
- 測試執行與除錯技巧

## 📝 使用方式

根據需求選擇對應的指南：
- 需要進行資料庫變更時 → 參考 DATABASE-MIGRATION-GUIDE.md
- 忘記 Maven 命令時 → 查看 DEVELOPMENT-TOOLS-GUIDE.md
- 設定新開發環境時 → 參考 DEVELOPMENT-TOOLS-GUIDE.md 的 IDE 設定
